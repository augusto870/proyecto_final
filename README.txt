# ACE Ingeniería & Conectividad (Proyecto estudiantil)

Este proyecto es un punto de partida simple para un sitio institucional. Incluye:
- `index.html` (estructura HTML5 con secciones: Inicio, Nosotros, Servicios, Proyectos, Contacto, Soporte)
- `css/estilos.css` (estilos básicos y responsivo por media queries)
- `js/script.js` (validación muy básica del formulario)
- `php/enviar.php` (ejemplo de envío por mail, DEMO)
- Carpeta `imagenes` con un logo y una imagen de proyecto de ejemplo (placeholders)

## Cómo usar en Cursor / VS Code
1. Abrí la carpeta del proyecto.
2. Usa una extensión de *Live Server* o servilo con un hosting simple (si usás PHP, necesitás servidor con PHP).
3. Cambiá en `php/enviar.php` el email de destino por uno real.
4. Reemplazá `imagenes/logo.png` y `imagenes/proyecto1.jpg` por tus propias imágenes.

> Nota: Los íconos (Font Awesome) cargan desde CDN, por lo que requieren internet.
